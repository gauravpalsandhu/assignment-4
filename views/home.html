Welcome to my website!

I will be demonstrating HTML5 principles and techniques.

Overview

This site consists of three static views and five dynamic routes:

  • Home - An overview of the site and HTML5 introduction.
  • About - A page with details about the creator.
  • HTML Demo - A demo page.
  • Single Student by Student Number - View details of a specific student (Student ID 2).
  • All Managers - All managers.
  • All Students - List of all students.
  • All Courses - List of all courses.
  • Students by Course - View students from a specific course (e.g., Course ID 3).

Introduction to HTML5

HTML5 is the latest version of Hypertext Markup Language, the standard for structuring and presenting content on the web. This website showcases HTML5 features, such as semantic elements, multimedia integration (audio, video), and responsive design through Bootstrap.
